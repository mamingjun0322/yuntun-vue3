## 🔌 后端接口对接

### 接口基础地址
- 开发环境：`http://localhost:8080`
- 生产环境：在 `.env.production` 中配置

### API接口列表

#### 认证相关
```
POST /admin/auth/login      # 管理员登录
GET  /admin/auth/info       # 获取管理员信息
POST /admin/auth/logout     # 退出登录
```

#### 商品管理
```
GET    /admin/goods/list       # 获取商品列表
POST   /admin/goods/add        # 添加商品
PUT    /admin/goods/update/:id # 更新商品
DELETE /admin/goods/delete/:id # 删除商品
PUT    /admin/goods/status/:id # 更新商品状态
```

#### 订单管理
```
GET /admin/order/list          # 获取订单列表
PUT /admin/order/status/:id    # 更新订单状态
GET /admin/order/statistics    # 获取订单统计
```

#### 用户管理
```
GET /admin/user/list           # 获取用户列表
```

#### 分类管理
```
GET /api/category/list         # 获取分类列表
```

## 🚀 启动步骤

### 1. 启动后端服务
```bash
cd ../yuntun-java
mvn spring-boot:run
```
确保后端运行在 http://localhost:8080

### 2. 启动前端项目
```bash
cd yuntun-vue3
npm install  # 如果还没安装依赖
npm run dev
```

### 3. 访问系统
打开浏览器访问：http://localhost:5173

### 4. 登录
- 用户名：admin
- 密码：admin123

## 🔧 开发技巧

### 1. Mock数据
在API请求失败时，代码会自动使用Mock数据，方便开发调试。

### 2. 环境变量
```bash
# .env.development
VITE_API_BASE_URL=http://localhost:8080

# .env.production
VITE_API_BASE_URL=https://your-api-domain.com
```

### 3. 代码规范
项目已配置ESLint和Prettier，运行以下命令格式化代码：
```bash
npm run lint    # 检查和修复代码
npm run format  # 格式化代码
```

### 4. 类型检查
```bash
npm run type-check  # TypeScript类型检查
```

## ⚠️ 注意事项

1. **后端依赖**
   - 必须先启动后端服务才能正常使用所有功能
   - 未连接后端时会使用Mock数据

2. **浏览器兼容**
   - 推荐使用Chrome、Edge等现代浏览器
   - 需要支持ES6+特性

3. **开发环境**
   - Node.js 版本需要 20.19.0+ 或 22.12.0+
   - 建议使用VS Code开发

4. **状态持久化**
   - 登录状态会保存在localStorage
   - 清除浏览器缓存会清除登录状态

## 📞 技术支持

如遇到问题，请检查：
1. Node.js版本是否符合要求
2. 依赖是否正确安装（删除node_modules重新安装）
3. 后端服务是否正常运行
4. 浏览器控制台错误信息
5. 网络请求是否正常
